### Configuration
- [Geany, highlight word](https://plugins.geany.org/automark.html): select `automark` plugin
- [Show whitespace](https://github.com/editorconfig/editorconfig/wiki/Property-research:-Trim-trailing-spaces): View > Editor > Show White Space option
- [Trim whitespace](https://github.com/editorconfig/editorconfig/wiki/Property-research:-Trim-trailing-spaces): Edit > Preferences > Files > Saving files > Strip trailing spaces and tabs
